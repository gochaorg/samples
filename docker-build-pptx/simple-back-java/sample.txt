OverlayFS - Файловая система Linux (одна из)
Монтирует файлы послойно
Нижние слои доступны для чтения, верхний для записи
Стопка слоев представляет содержание образа
Альтернативы/варианты docker storage driver
overlay2 - это overlayfs
fuse-overlayfs - это overlayfs в пространстве пользователя
btrfs, zfs - расширенные файловые системы, которые можно использовать самостоятельно
vfs - специфичная реализация, не рекомендовано, не умеет Copy-On-Write
devicemapper - это реализация на базе менеджера томов LVM2, рекомендовано для CentOS и RHEL, ибо они не умеют в overlay2 (сейчас умеют)
